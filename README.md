# 💰 Epoint System

**一个简单的积分商店插件**

([https://github.com/CandyourGoose/EpointPlugin](https://github.com/CandyourGoose/EpointPlugin]))



## 📖 项目简介

**EasyPoint** 是一个为 TShock 服务器设计的经济系统，引入了**自动签到**、**在线奖励**、**BOSS奖励**、**日常刷怪**等积分奖励方法，并设计了对应的**积分商店**以供消费。

无论是前期爆肝开荒，还是后期休闲养老，Epoint 都能为服务器增添趣味与提供帮助。

-----

## ✨ 核心特性 (Features)

  * 🛡️ **双重并发检验签到**：结合 `底层心跳扫描` 与 `事件驱动拦截`，实现全自动防漏签。
  * 📈 **动态天数与防通胀上限**：玩家每日获取的积分上限会随着服务器游玩天数动态增长，配合 `Math.Min` 上限截断机制锁死后期数值膨胀。
  * ⚔️ **Boss 击杀奖励分配**：引入非线性平滑函数兼容高血量 Mod/事件 Boss。支持【按玩家伤害贡献分发奖励】或【全服在线玩家平均分发奖励】双模式。
  * 🦇 **小怪击杀里程碑奖励**：每累计击杀 50 只同类怪物即可获得里程碑奖励，日常跑图清怪依然能获得正向多巴胺反馈。
  * 🛒 **内置 UI 商店**：内置聊天框商店 UI，支持商品堆叠配置与【限购道具】验证。
  * 🎲 **拟真运气系统**：签到奖励融合了随机运气波动机制，并配有专属的颜文字交互提示Ciallo～ (∠・ω< )⌒★。

## ⌨️ 玩家指令 (Commands)

| 指令语法 | 权限 | 描述说明 |
| :--- | :--- | :--- |
| `/ephelp` | 默认 | 呼出 Epoint 积分系统的帮助菜单 |
| `/epinfo` | 默认 | 查看玩家个人的积分余额、今日额度、累积天数与连签状态 |
| `/epshop` | 默认 | 打开积分商店，查看当前上架的普通/限购商品列表 |
| `/epbuy <商品序号> [数量]` | 默认 | 购买指定序号的商品（不输入数量则默认为 1 份） |
| `/epreload` | `epoint.admin` | **[管理员]** 热重载配置文件与定时器，无需重启服务器 |

## ⚙️ 配置文件 (Configuration)

首次加载插件后，会在 `tshock/epoint/` 目录下自动生成 `epointconfig.json` 文件。支持热修改并使用 `/epreload` 实时生效。

```json
{
  "快节奏模式(设为true则积分获取效率及上限翻倍, 适用于进度推进快的高手玩家，默认false)": false,
  "每日积分获取基础上限(默认1000)": 1000,
  "在线奖励积分(默认5)": 5,
  "在线奖励间隔(分钟, 默认10)": 10,
  "Boss奖励积分分配方式(按贡献分配true|平均分配false, 默认true)": true,
  "商品列表": [
    {
      "序号": 1,
      "商品名称": "强效治疗药水",
      "原版物品ID": 499,
      "积分价格": 20,
      "物品数量": 5,
      "是否限购": false
    },
    {
      "序号": 3,
      "商品名称": "史莱姆法杖",
      "原版物品ID": 1309,
      "积分价格": 5000,
      "物品数量": 1,
      "是否限购": true
    }
  ]
}
```

> **💡 提示：** “是否限购”设为 `true` 的商品，玩家只能购买一次，其序号在游戏内商店中自动标红。

## 💾 数据存储机制

本插件采用了**独立玩家 JSON 文件**进行存储管理。

  * 路径：`tshock/epoint/players/<玩家名>.json`
  * 服主可随时查阅或修改玩家的积分与限购记录。
  * 数据读写与内存解耦，修改文件后玩家的下一次积分变动将直接继承新数据。

## 📦 安装指南 (Installation)

1.  下载最新版本的 `EpointPlugin.dll` 文件（`EpointPlugin.pdb` 文件可选）。
2.  将该 `.dll` （`.pdb` 可选）文件放入泰拉瑞亚服务器的 `ServerPlugins` 文件夹中。
3.  重启服务器（或使用 /reload 指令）。
4.  进服输入 `/ephelp` 验证是否加载成功。
